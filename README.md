# GoatAbstract

GoatAbstract is a dark-mode-only Vite website for generating seeded abstract wave line art. It is built for placeholder images, filler artwork, mockups, and other exact-size visual blocks.

## Features

- Exact width and height controls
- Deterministic seed input for reproducible outputs
- Custom line color and background color
- Static SVG preview
- Download actions for SVG and PNG

## Development

```bash
npm install
npm run dev
```

## Validation

```bash
npm run test -- --run
npm run lint
npm run build
```
